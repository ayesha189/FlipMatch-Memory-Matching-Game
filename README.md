# 🧠 FlipMatch — Memory Matching Game

A fun, browser-based memory card matching game with multiple difficulty levels, themed emoji cards, sound effects, and smooth 3D flip animations. No frameworks, no dependencies — just open `index.html` and play!

---

## ✨ Features

- 🎯 Three difficulty levels — Easy, Medium, Hard
- 🎴 Themed emoji sets — Fruits, Animals, and Mixed
- ⏱️ Live move counter and timer
- 🔊 Sound effects for matches, misses, and winning
- 🃏 Smooth 3D card flip with shake animation on wrong pairs
- 📱 Fully responsive — works on mobile and desktop

---

## 🕹️ How to Play

1. Choose a difficulty from the main menu
2. Click any card to flip it and reveal the emoji
3. Click a second card — if they match, they stay face-up ✅
4. If they don't match, they flip back after a short delay ❌
5. Match all pairs to win!

---

## 🎯 Difficulty Levels

| Level  | Pairs | Grid  | Theme      |
|--------|-------|-------|------------|
| Easy   | 8     | 4 × 4 | 🍎 Fruits  |
| Medium | 10    | 4 × 5 | 🐶 Animals |
| Hard   | 12    | 4 × 6 | 🎯 Mixed   |

## 🚀 Getting Started

No installation needed. Just clone and open:

```bash
git clone https://github.com/your-username/flipmatch.git
cd flipmatch
open index.html
```

> **Note:** Place `wow.mp3`, `oops.mp3`, and `party.mp3` in the same directory as `index.html` for sound effects to work.
